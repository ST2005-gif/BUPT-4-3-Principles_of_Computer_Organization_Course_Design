# 北邮计科大二下课程——计算机组成原理课程设计
## 2025年春
### 课程设计简介

该课程要求设计实现基于时序信号控制的流水硬布线控制器，在TEC-8实验台上实现类似于微程序控制器的功能

项目为四人项目

### 项目设定
使用语言：Verilog

编译平台：Quartus II 9.01

### 运行方式
启动Quartus II 9.01，将项目文件夹添加至项目文件（选中.qpf文件即可添加），随后接好线路，编译下载到TEC-8实验台即可
